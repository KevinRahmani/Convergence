----------------------------FICHIER README------------------------------

Le programme regroupe l'ensemble des fonctions des exercices organisés autour d'un menu.

Veuillez vous assurer que vous avez gnuplot de télécharger sur votre ordinateur pour le bon fonctionnement du programme : si ce n'est pas le cas veuillez l'installer avec la ligne de commande suivante : sudo apt-get install gnuplot sous Ubuntu

Une fois l'installation faite, rendez-vous sur le dossier Modele_proba avec un cd , ensuite veuillez donner le droit d'éxécution à main.sh avec un : chmod +x main.sh dans le terminal ou sudo chmod +x si vous n'êtes pas propriétaire de la machine et éxécutez le : ./main.sh (les programmes C se compileront tout seuls). Répétez autant de fois que vous voulez, les résultats des calculs se trouveront dans le fichier use.dat et les graphiques dans le fichier, le tout dans le dossier result.
